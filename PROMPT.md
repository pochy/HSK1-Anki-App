# AI Conversation Data Generation Prompt

このファイルは、AI（ChatGPT, Claude, Geminiなど）を使用して「Conversation Simulator」用の会話シーンデータを生成するためのプロンプトです。この内容をコピーしてAIに入力してください。

---

# Role
あなたは中国語教育の専門家で、語学学習アプリのコンテンツ作成者です。

# Task
中国語会話シミュレーションゲーム「Conversation Simulator」のための会話シーンデータ（JSON）を作成してください。

# Target Audience
- 日本人の中国語学習者
- レベル: HSK 1級〜6級

# Vocabulary Constraints & Level Definitions (重要)
作成するシーンの難易度に合わせて、以下の基準を厳格に守ってください。

## HSK 1級 (Difficulty 1)
- **語彙**: 約150語（累積）
- **トピック**: 挨拶、自己紹介、家族、国籍、数字、日付、時間、簡単な買い物
- **能力**: 非常にシンプルで短いフレーズ。文法はSVOの基本形が中心。
- **制約**: 150語の範囲外の単語は極力避けること。

## HSK 2級 (Difficulty 2)
- **語彙**: 約300語（累積）
- **トピック**: 日常生活、趣味、天気、道案内、注文、簡単な感情表現
- **能力**: 複文（因为...所以...など）が少しずつ登場。日常会話の簡単な受け答えが可能。

## HSK 3級 (Difficulty 3)
- **語彙**: 約600語（累積）
- **トピック**: 生活、学習、仕事、旅行、健康、娯楽、交通機関の利用
- **能力**: 比較表現、補語（方向補語、可能補語など）の使用。中国旅行で困らない程度の会話力。

## HSK 4級 (Difficulty 4)
- **語彙**: 約1,200語（累積）
- **トピック**: 広範囲な日常話題、社会問題の入り口、環境、文化
- **能力**: 複雑な文法構造（把構文、被構文など）の自然な使用。ネイティブとの比較的流暢な会話。

## HSK 5級 (Difficulty 5)
- **語彙**: 約2,500語（累積）
- **トピック**: 抽象的な話題、ニュース、映画、ビジネス詳細、スピーチ
- **能力**: 成語の使用。よりフォーマルな表現や書き言葉的な表現の理解と使用。

## HSK 6級 (Difficulty 6)
- **語彙**: 5,000語以上（累積）
- **トピック**: 専門的な議論、文学、歴史、科学、時事問題の深い分析
- **能力**: ネイティブレベルの表現力。微妙なニュアンスの使い分け、ユーモアや皮肉の理解。

# Recommended Scenes List (シーン作成リスト)
データを生成する際は、以下のリストからシーンを選択してください。

## HSK 1級
1. **基本的な挨拶**: 「ニーハオ」から始まる初対面の会話
2. **名前を尋ねる**: 自分の名前を言い、相手の名前を聞く
3. **国籍と出身**: 「あなたはどこの国の人ですか？」
4. **数字と年齢**: 子供に年齢を聞く、電話番号を伝える
5. **日時を尋ねる**: 「今日は何月何日ですか？」「何曜日ですか？」
6. **家族の紹介**: 写真を見せながら家族構成を話す
7. **場所を尋ねる**: 「トイレはどこですか？」「駅はどこですか？」
8. **カフェでの注文**: お茶やコーヒーを頼む（最もシンプルな形）
9. **市場での買い物**: 果物の値段を聞いて買う
10. **持ち物の確認**: 「これはあなたの本ですか？」

## HSK 2級
1. **レストランでの注文**: メニューを見て料理を注文する
2. **道案内**: 「まっすぐ行って右に曲がってください」
3. **天気の話**: 「明日は雨が降りますか？」「とても暑いです」
4. **趣味の話**: 「週末は何をしますか？」「映画を見るのが好きです」
5. **衣服の買い物**: 色やサイズについて店員と話す
6. **電話をかける**: 友達を呼び出す、不在の伝言
7. **食事への招待**: 友達を晩ご飯に誘う
8. **チケット購入**: 電車や映画のチケットを買う
9. **体調不良**: 医者にお腹が痛いと伝える
10. **一日の予定**: 何時に起きて、何時に寝るか話す

## HSK 3級
1. **ホテルのチェックイン**: 予約の確認と部屋の要望
2. **飛行機の予約変更**: 時間の変更を依頼する
3. **部屋探し**: 家賃や条件について大家と話す
4. **週末の感想**: 週末に行った旅行の経験を語る（過去形・完了）
5. **比較する**: 2つの商品のどちらが良いか比べる
6. **贈り物を渡す**: 誕生日プレゼントを渡し、お祝いを言う
7. **遅刻の理由**: なぜ遅れたのか状況を説明する
8. **詳細な道案内**: バスや地下鉄の乗り換えを含めた説明
9. **商品の返品**: 壊れていた商品を返品・交換する
10. **手伝いを頼む**: 宿題や仕事の手伝いを依頼する

## HSK 4級
1. **就職面接**: 自己紹介と長所・短所のアピール
2. **環境保護**: ゴミの分別やエコ活動について話す
3. **誤解を解く**: 友達との喧嘩や誤解を解決する
4. **ネットショッピング**: 配送トラブルやレビューについて話す
5. **文化紹介**: 自国の祭りや習慣を中国人の友人に紹介する
6. **社会問題への意見**: 交通渋滞やスマホ依存について議論する
7. **励ましとアドバイス**: 落ち込んでいる友人を励ます
8. **映画・本の内容**: あらすじと感想を詳しく語る
9. **計画の変更と交渉**: 旅行の日程変更を友人と相談する
10. **将来の夢**: 具体的なキャリアプランや目標を語る

## HSK 5級
1. **経済ニュース**: 最近の景気動向について議論する
2. **ビジネス交渉**: 価格交渉や契約条件の詰め
3. **伝統文化の解説**: 京劇や書道などの深い歴史的背景を説明する
4. **結婚式のスピーチ**: 新郎新婦への祝辞を述べる
5. **クレーム対応**: 店長として顧客の苦情に丁寧に対応する
6. **抽象的な芸術**: 絵画や現代アートの印象を語り合う
7. **人生観・価値観**: 幸せとは何か、成功とは何かを語る
8. **故事成語の使用**: 会話の中で適切な成語を使って例える
9. **企業の戦略会議**: 新製品のマーケティング戦略を話し合う
10. **最新技術の批評**: AIや宇宙開発の是非について議論する

## HSK 6級
1. **文学作品の批評**: 古典文学や現代小説のテーマを深く分析する
2. **歴史的事件**: 歴史上の出来事の要因と影響を論じる
3. **国際情勢の議論**: 国家間の関係や外交問題について意見を交わす
4. **皮肉とユーモア**: 高度な言い回しやジョークを含んだ会話
5. **哲学的な対話**: 人間の存在意義や倫理について語る
6. **科学技術と倫理**: 遺伝子操作などの倫理的問題を議論する
7. **詩の解釈**: 中国の詩（唐詩など）の意味と美しさを解説する
8. **法律相談**: 複雑な事件や法律問題について専門的に話す
9. **方言と地域文化**: 方言の違いや地域ごとの気質について語る
10. **即興スピーチ**: 与えられたテーマで説得力のあるスピーチをする

# Specifications
以下のTypeScriptインターフェース（仕様）に厳密に従ってJSONデータを作成してください。
各シーンには必ず **4つの選択肢（options）** を含め、そのうち1つを正解、3つを不正解にしてください。

## Data Structure
```typescript
interface ConversationScene {
  scene_id: number; // ユニークな整数ID
  title: string; // シーンのタイトル（例: "タクシーに乗る"）
  description: string; // 日本語での状況説明
  category: SceneCategory;
  difficulty_level: 1 | 2 | 3 | 4 | 5 | 6; // HSKレベルに対応
  context: {
    setting: string; // 場所・状況
    participants: { role: string; name?: string }[];
    relationship: "friend" | "stranger" | "superior" | "colleague" | "family";
    formality_level: "casual" | "polite" | "formal";
    emotional_tone: "neutral" | "friendly" | "apologetic" | "grateful" | "urgent";
  };
  dialogue: {
    speaker: string;
    text: string; // 中国語
    pinyin: string;
    translation: string; // 日本語
    order: number;
  }[];
  options: {
    response_id: string; // "r1", "r2", "r3", "r4"
    text: string; // 中国語
    pinyin: string;
    translation: string; // 日本語
    is_correct: boolean;
    rating: "perfect" | "good" | "ok" | "poor";
    explanation: string; // なぜ正解/不正解なのか、文法や文化的な解説
    distractor_type?: "grammar_error" | "wrong_context" | "too_formal" | "too_casual" | "wrong_meaning";
  }[];
  grammar_points: {
    point_id: string;
    title: string;
    description: string;
    examples: {
      chinese: string;
      pinyin: string;
      japanese: string;
      explanation: string;
    }[];
    related_scenes: number[];
  }[];
  // 以下のメタデータはデフォルト値で埋めてください
  srs_data: {
    next_review: string; // Now (ISO string)
    interval: 0;
    stability: 0;
    difficulty: 0;
    streak: 0;
    review_count: 0;
  };
  metadata: {
    created_date: string; // Now (ISO string)
    total_plays: 0;
    perfect_count: 0;
    category_color: string; // カテゴリに合った色 (Hex)
    estimated_time: number; // 推定秒数 (例: 30)
  };
}

type SceneCategory = "greeting" | "shopping" | "restaurant" | "transportation" | "asking_directions" | "phone" | "friendship" | "business" | "emergency" | "daily_life";
```

## Requirements for Options (選択肢の要件)
各シーンの `options` は必ず以下の構成にしてください：

1. **正解 (Perfect)**
   - `is_correct: true`, `rating: "perfect"`
   - 文脈に最も適した自然な表現。

2. **不正解パターンA (Wrong Context / Meaning)**
   - `is_correct: false`
   - 文法は正しいが、文脈に合わない（例：飲食店で「道を教えて」と言う）、または意味が異なる。

3. **不正解パターンB (Too Casual / Too Formal)**
   - `is_correct: false`
   - 文法や意味は通じるが、相手との関係性に適さない（例：店員にタメ口、友達に堅苦しい敬語）。

4. **不正解パターンC (Grammar Error / Poor)**
   - `is_correct: false`, `rating: "poor"`
   - 初心者がやりがちな文法ミスや不自然な直訳。

# Output Format
- 各シーンは **個別のJSONファイル** として出力してください。
- ファイル名は `hsk[レベル]_[シーン番号]_[シーン名].json` の形式（例: `hsk1_01_greeting.json`）にしてください。
- 各ファイルの内容は、定義した `ConversationScene` オブジェクト **単体** である必要があります（配列で囲まないでください）。
- 解説（explanation, description）はすべて **日本語** で記述してください。
- 出力する際は、コードブロックの冒頭にファイル名をコメントとして記載してください。
