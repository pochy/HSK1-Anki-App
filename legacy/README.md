# 新 HSK1 級 スピードマスター 🇨🇳

[![HSK Level](https://img.shields.io/badge/HSK-Level%201-orange)](https://en.wikipedia.org/wiki/Hanyu_Shuiping_Kaoshi)
[![Words](https://img.shields.io/badge/単語数-428語-blue)](./index.html)
[![License](https://img.shields.io/badge/license-MIT-green)](./LICENSE)

新 HSK1 級（2021 年改訂版）に対応した、モバイルフレンドリーな中国語単語学習 Web アプリケーションです。

## ✨ 特徴

- **📚 完全版 428 語収録** - HSK1 級指定語句 150 語 + 頻出・生活語句 278 語
- **🎯 カテゴリー別学習** - 品詞・テーマごとに効率的に学習
- **🎮 クイズモード** - ランダム 10 問の総合実力テスト
- **📋 単語リスト** - カテゴリー別に全単語を一覧表示
- **🔊 音声読み上げ** - Web Speech API によるネイティブ発音
- **💾 進捗保存** - ローカルストレージで学習記録を保持
- **📱 モバイル対応** - スマートフォンでも快適に学習可能
- **🧭 固定ナビゲーション** - ページ下部のタブで主要機能に素早くアクセス

## 🚀 使い方

### オプション 1: ローカルで開く

```bash
# リポジトリをクローン
git clone https://github.com/pochy/HSK1-Anki-App.git

# フォルダに移動
cd HSK1-Anki-App

# index.htmlをブラウザで開く
open index.html  # macOS
start index.html # Windows
```

### オプション 2: Web サーバーで配信

```bash
# Python 3 の場合（http://localhost:8000）
python -m http.server 8000

# Node.js の場合（http://localhost:3000）
npx serve .
```

起動後に表示される URL（例: `http://localhost:8000` / `http://localhost:3000`）へアクセスしてください。

## 🎓 学習モード

### 1. カテゴリー学習

各カテゴリーをタップして、フラッシュカード形式で単語を学習します。

- **覚えた** → 習得済みとしてマーク
- **まだ** → 未習得として復習リストに保持

### 2. 苦手復習

未習得の単語からランダムに 10 語を選んで集中学習します。

### 3. 総合実力テスト

全 428 語からランダムに 10 問出題されるクイズモードです。

- 音声 → 意味
- 漢字 → 意味
- 意味 → 漢字

### 4. 単語リスト

カテゴリー別に全単語を一覧表示します。

- カテゴリーごとにグループ化
- 各単語の習得状況を表示
- 音声読み上げボタン付き
- カテゴリーごとの進捗率表示

## 🧭 ナビゲーション

## 🔧 カスタマイズ

### 単語データの編集

`index.html` 内の `hskData` 配列を編集することで、単語の追加・変更が可能です：

```javascript
const hskData = [
  {
    id: 101, // ユニークID
    char: "我", // 中国語（簡体字）
    pinyin: "wǒ", // ピンイン
    meaning: "私", // 日本語訳
    category: "代名詞", // カテゴリー
    example: "我是学生。", // 例文
    exampleMeaning: "私は学生です。", // 例文訳
  },
  // ...
];
```

### スタイルのカスタマイズ

CSS 変数を編集してカラーテーマを変更できます：

```css
:root {
  --primary: #f59e0b; /* メインカラー（アンバー） */
  --primary-dark: #d97706;
  --secondary: #3b82f6; /* セカンダリカラー（ブルー） */
  --success: #22c55e;
  --danger: #ef4444;
}
```
